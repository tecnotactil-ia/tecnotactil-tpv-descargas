# Escritorio — TecnoTactil Negocio (Windows)

Sube aquí el instalador de la app de **escritorio** con el **nombre EXACTO**:

**`TecnoTactil_Negocio.exe`**

La web enlaza directamente a:
`https://github.com/tecnotactil-ia/tecnotactil-tpv-descargas/raw/main/escritorio/TecnoTactil_Negocio.exe`

El enlace de la web sigue funcionando solo. (La versión vieja PyQt en *Releases* se puede borrar cuando esta esté arriba.)
